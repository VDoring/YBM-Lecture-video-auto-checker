## ✔️ 작동원리
프로그램이 켜져 있는 동안 컴퓨터 화면에서 체크포인트 창의 '예' 버튼이 있는지 지속적으로 확인합니다. 체크포인트 창이 나타나면 프로그램이 자동으로 '예'버튼을 누릅니다.
